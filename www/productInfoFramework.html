Alfresco Repository

The Alfresco repository provides out-of-the-box portal integration and full content control with integrated document management, security, document status and workflow.

Alfresco Web Scripts

Alfresco Web Scripts provide an REST-style API to the Alfresco Repository. Further information can be found here: Alfresco Web Scripts Technical Documentation

Ext JS Framework

Ext JS (Product Website) is a cross-browser JavaScript library for building rich internet applications, available on a dual-license basis - either under the GNU GPL license v3 or a commercial license. DoCASU is using version 2.0 of Ext JS.

FreeMarker Templates

FreeMarker is a Java Template Engine Library: Freemarker Project Website DoCASU is making use of the bundled FreeMarker capabilities of Alfresco.
